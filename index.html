Install and safety note

  • VICA is designed as a driver-assistance aid, not a replacement for judgment.
  • Validate behavior first in controlled sessions before any race event use.
  • Keep a direct stop/exit path available for immediate disable or quick disable.
  • Use conservative visual settings and disable VICA if you lose focus.
